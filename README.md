# For Linux/Unix

## Install these tools

- [A Nerdfont (CaskaydiaCove NF Mono)](https://www.nerdfonts.com/font-downloads)
- zsh
- tmux
- bat
- fzf
- btop
- oh-my-posh
- zoxide
- eza

## tmux

- Install tmux
- Install TPM `git clone https://github.com/tmux-plugins/tpm ~/.tmux/plugins/tpm`
- Prefix + I (To install TPM plugins)
- Reload .tmux.conf

## Oh-my-posh
* Linux `curl -s https://ohmyposh.dev/install.sh | bash -s`
* macOS `brew install jandedobbeleer/oh-my-posh/oh-my-posh`

# For Windows

## Core Basics
- `winget install Microsoft.PowerShell`
- `winget install Microsoft.WindowsTerminal`

## Install OhMyPosh and Font (CaskaydiaCove NF)
- `winget install JanDeDobbeleer.OhMyPosh -s winget`
- `oh-my-posh font install cantarell`

Note: Oh-my-posh font installer is the easiest way. 
Select 'CaskaydiaCove' from the list it presents.

## Install Tools
- `winget install btop`
- `winget install sharkdp.bat`
